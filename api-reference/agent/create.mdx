---
title: 'Create Persona'
api: 'POST /persona'
description: 'Create a new persona'
authMethod: "bearer"
---

This endpoint creates a new persona for the authenticated user's account.

### Body

<ParamField body="name" type="string" required>
  The name of the persona to be created.
</ParamField>

### Response

<ResponseField name="persona" type="object">
  The created persona object.
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.privvy.xyz/persona' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_API_KEY' \
-d '{"name": "My New Persona"}'
```
</RequestExample>

<ResponseExample>
```
json Response
{
  "agent": {
    "id": "60a3e5b9f3f3a2001f3e8d1a",
    "name": "My New Persona",
    "account": "60a3e5b9f3f3a2001f3e8d1b",
    "createdBy": "60a3e5b9f3f3a2001f3e8d1c",
    "createdAt": "2023-05-18T12:00:00.000Z",
    "updatedAt": "2023-05-18T12:00:00.000Z"
  }
}
```
</ResponseExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if the persona creation fails.
</ResponseField>

- HTTP 403: Returned if the user doesn't have the required subscription or has reached the agent limit.
- HTTP 409: Returned if a persona with the same name already exists.
- HTTP 500: Returned if there's a server error during persona creation.
