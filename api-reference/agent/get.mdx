---
title: 'Get Agents'
api: 'GET /agent'
description: 'Gets agents for the authenticated user'
authMethod: "bearer"
---

This endpoint loads agents for the authenticated user's account.

### Query Parameters

<ParamField query="name" type="string">
  Optional. Filter agents by name.
</ParamField>

### Response

<ResponseField name="agents" type="array">
  An array of agent objects.
</ResponseField>

<RequestExample>
```bash
curl 'https://api.privvy.xyz/agent' \
-H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving agents fails.
</ResponseField>

- HTTP 500: Returned if there's a server error while retrieving agents.
