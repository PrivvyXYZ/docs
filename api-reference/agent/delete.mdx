---
title: 'Delete Agent'
api: 'DELETE /agent'
description: 'Delete a agent and its associated memories'
authMethod: "bearer"
---

This endpoint deletes a agent and its associated memories for the authenticated user's account.

### Body

<ParamField body="_id" type="string" required>
  The ID of the agent to be deleted.
</ParamField>

### Response

<ResponseField name="agent" type="object">
  The deleted agent object.
</ResponseField>

<RequestExample>
```bash
curl -X DELETE 'https://api.privvy.xyz/agent' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_API_KEY' \
-d '{"id": "60a3e5b9f3f3a2001f3e8d1a"}'
```
</RequestExample>

<ResponseExample>
```json
{
  "agent": {
    "id": "60a3e5b9f3f3a2001f3e8d1a",
    "isArchived": true
  }
}
```
</ResponseExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if the agent deletion fails.
</ResponseField>

- HTTP 500: Returned if there's a server error during agent deletion.

<Note>
  This operation also archives all associated memories and deletes related vectors.
</Note>
