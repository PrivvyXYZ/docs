This endpoint retrieves zones for the authenticated user's account.

### Query Parameters

<ParamField query="name" type="string">
  Optional. Filter zones by name.
</ParamField>

### Response

<ResponseField name="zones" type="array">
  An array of zone objects.
</ResponseField>
