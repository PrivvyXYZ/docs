---
title: 'Get Zones'
api: 'GET /zone'
description: 'Retrieve zones for the authenticated user'
---

This endpoint retrieves zones for the authenticated user's account.

### Query Parameters

<ParamField query="name" type="string">
  Optional. Filter zones by name.
</ParamField>

### Response

<ResponseField name="zones" type="array">
  An array of zone objects.
</ResponseField>

<RequestExample>
```bash
curl 'https://api.privvy.xyz/zone' \
-H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving zones fails.
</ResponseField>

- HTTP 500: Returned if there's a server error while retrieving zones.
