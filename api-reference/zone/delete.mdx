---
title: 'Delete Zone'
api: 'DELETE /zone'
description: 'Delete a zone and its associated sources'
---

This endpoint deletes a zone and its associated sources for the authenticated user's account.

### Body

<ParamField body="_id" type="string" required>
  The ID of the zone to be deleted.
</ParamField>

### Response

<ResponseField name="zone" type="object">
  The deleted zone object.
</ResponseField>

<RequestExample>
```bash
curl -X DELETE 'https://api.yourdomain.com/zone' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_API_KEY' \
-d '{"id": "60a3e5b9f3f3a2001f3e8d1a"}'
```
</RequestExample>

<ResponseExample>
```json
{
"zone": {
"id": "60a3e5b9f3f3a2001f3e8d1a",
"isArchived": true
}
}
```
</ResponseExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if the zone deletion fails.
</ResponseField>

- HTTP 500: Returned if there's a server error during zone deletion.

<Note>
  This operation also archives all associated sources and deletes related vectors.
</Note>
