---
title: 'Get Sources'
api: 'GET /source'
description: 'Retrieve sources for the authenticated user'
authMethod: "bearer"
---

This endpoint retrieves sources for the authenticated user's account.

### Query Parameters

<ParamField query="name" type="string">
  Optional. Filter sources by name.
</ParamField>

### Response

<ResponseField name="sources" type="array">
  An array of source objects.
</ResponseField>

<RequestExample>
```bash
curl 'https://api.privvy.xyz/source' \
-H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving sources fails.
</ResponseField>

- HTTP 500: Returned if there's a server error while retrieving sources.
