---
title: 'Create URL Source'
api: 'POST /source/url'
description: 'Create a new source from a URL'
authMethod: "bearer"
---

This endpoint creates a new source for the authenticated user's account from a URL. It tests the URL, identifies the source type, and initiates the document processing.

### Authorization

<Authorization name="Authorization" type="string" required>
  The API key to use for authentication. Must have WRITE_SOURCES permission.
</Authorization>

### Body

<ParamField body="url" type="string" required format="uri">
  The URL of the website to create a source from.
</ParamField>
<Note>
  Depending on the size of the website and the number of pages, it may take some time to process the source.
  The maximum number of pages that can be extracted is 500.
</Note>

<ParamField body="autoUpdate" type="boolean" required>
  Whether to automatically update the source at a 24 hour interval.
</ParamField>

<ParamField body="metadata" type="object" optional>
  Metadata to store with the source.
</ParamField>

<ParamField body="agentId" type="string" optional>
  The ID of the agent to create the source in.
</ParamField>

<ParamField body="agentReferenceId" type="string" optional>
  The reference ID of the agent to create the source in.
</ParamField>

Either `agentId` or `agentReferenceId` must be provided.

### Response

<RequestExample>
```bash
curl -X POST 'https://api.privvy.xyz/source/url' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_API_KEY' \
-d '{"url": "https://example.com", "autoUpdate": true, "metadata": {"key": "value"}, "agentId": "agent123"}'
```
</RequestExample>

<ResponseExample>
```json
{
  "source": {
    "id": "60a3e5b9f3f3a2001f3e8d1a",
    "url": "https://example.com",
    "autoUpdate": true,
    "type": "WebPage",
    "settingParams": {
      "metadata": {"key": "value"},
      "maxCrawlable": 500
    }
  },
  "message": "Source created successfully"
}
```
</ResponseExample>

<ResponseField name="source" type="object">
  The created source object.
</ResponseField>

<ResponseField name="message" type="string">
  A success message.
</ResponseField>
