---
title: 'Create Plain Text Source'
api: 'POST /source/text'
description: 'Create a new source from plain text'
authMethod: "bearer"
---

This endpoint creates a new source for the authenticated user's account from provided plain text. It preprocesses the text, generates embeddings, and stores the content in our database.

### Body

<ParamField body="content" type="string" required>
  The plain text content to create a source from.
</ParamField>

<Note>
  The content must be more than 10 words and not exceed the maximum body length.
</Note>

<ParamField body="title" type="string" optional>
  The title of the source.
</ParamField>

<ParamField body="agentId" type="string" optional>
  The ID of the agent to create the source in.
</ParamField>

<ParamField body="agentReferenceId" type="string" optional>
  The reference ID of the agent to create the source in.
</ParamField>

### Response

<RequestExample>
```bash
curl -X POST 'https://api.privvy.xyz/source/text' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_API_KEY' \
-d '{"content": "Your plain text content here", "title": "Optional Title"}'
```
</RequestExample>

<ResponseExample>
```json
{
  "source": {
    "id": "60a3e5b9f3f3a2001f3e8d1a",
    "identifier": "abc123def456",
    "autoUpdate": false,
    "settingsType": "CUSTOM",
    "type": "PlainText",
    "status": "InProgress"
  },
  "message": "Text source created successfully"
}
```
</ResponseExample>

<ResponseField name="source" type="object">
  The created source object.
</ResponseField>

<ResponseField name="message" type="string">
  A success message.
</ResponseField>

<Note>
  Source creation is subject to account limits based on subscription type.
</Note>
