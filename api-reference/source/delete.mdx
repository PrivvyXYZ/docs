---
title: 'Delete Source'
api: 'DELETE /source'
description: 'Delete a source'
authMethod: "bearer"
---

This endpoint deletes a source for the authenticated user's account.

### Body

<ParamField body="_id" type="string" required>
  The ID of the source to be deleted.
</ParamField>

### Response

<ResponseField name="source" type="object">
  The deleted source object.
</ResponseField>

<RequestExample>
```bash
curl -X DELETE 'https://api.privvy.xyz/source' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_API_KEY' \
-d '{"id": "60a3e5b9f3f3a2001f3e8d1a"}'
```
</RequestExample>

<ResponseExample>
```json
{
  "source": {
    "id": "60a3e5b9f3f3a2001f3e8d1a",
    "isArchived": true
  }
}
```
</ResponseExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if the zone deletion fails.
</ResponseField>

- HTTP 500: Returned if there's a server error during zone deletion.

<Note>
  This operation also archives all associated sources and deletes related vectors.
</Note>
