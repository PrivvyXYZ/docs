---
title: 'Find a Source'
api: 'GET /source/:id'
description: 'Retrieve a specific source by its ID'
authMethod: "bearer"
---

Retrieve a specific source by its ID for the authenticated user's account.

### Path Parameters

<ParamField path="id" type="string" required>
  The ID of the source to retrieve.
</ParamField>

### Response

<ResponseField name="source" type="object">
  The source object with the following properties:
</ResponseField>

<ResponseField name="_id" type="string">
  The unique identifier of the source.
</ResponseField>

<ResponseField name="account" type="string">
  The account ID associated with the source.
</ResponseField>

<ResponseField name="type" type="string">
  The type of the source. Possible values: 'website', 'pdf', 'custom', 'github', 'gitbook', 'google_doc', 'discord_messages', 'confluence', 'notion', 'PlainText'.
</ResponseField>

<ResponseField name="identifier" type="string">
  A unique identifier for the source.
</ResponseField>

<ResponseField name="settingsType" type="string">
  The type of settings for the source. Possible values: 'WebsiteSourceSettings', 'PdfSourceSettings', 'CustomSourceSettings', 'GithubSourceSettings', 'GitbookSourceSettings', 'GoogleDocSourceSettings', 'DiscordMessagesSourceSettings', 'ConfluenceSourceSettings'.
</ResponseField>

<ResponseField name="autoUpdate" type="boolean">
  Indicates if the source is set to update automatically.
</ResponseField>

<ResponseField name="settings" type="string">
  The ID of the settings object associated with this source.
</ResponseField>

<ResponseField name="documentCount" type="number">
  The number of documents in this source.
</ResponseField>

<ResponseField name="agent" type="string">
  The ID of the agent this source belongs to.
</ResponseField>

<ResponseField name="isArchived" type="boolean">
  Indicates if the source has been archived.
</ResponseField>

<ResponseField name="chunkCount" type="number">
  The number of chunks in this source.
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the source. Possible values: 'InProgress', 'Completed', 'Failed', 'CompletedWithErrors', 'Preparing'.
</ResponseField>

<ResponseField name="createdAt" type="string">
  The timestamp when the source was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  The timestamp when the source was last updated.
</ResponseField>

<RequestExample>
```bash
curl 'https://api.privvy.xyz/source/60a3e5b9f3f3a2001f3e8d1a' \
-H 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving the source fails.
</ResponseField>

- HTTP 404: Returned if the source is not found.
- HTTP 500: Returned if there's a server error while retrieving the source.
