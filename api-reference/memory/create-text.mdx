---
title: 'Create Plain Text Memory'
api: 'POST /memory/text'
description: 'Create a new memory from plain text'
authMethod: "bearer"
---

This endpoint creates a new memory for the authenticated user's account from provided plain text. It preprocesses the text, generates embeddings, and stores the content in our database.

### Body

<ParamField body="content" type="string" required>
  The plain text content to create a memory from.
</ParamField>

<Note>
  The content must be more than 10 words and not exceed the maximum body length.
</Note>

<ParamField body="title" type="string" optional>
  The title of the memory.
</ParamField>

<ParamField body="personaId" type="string" optional>
  The ID of the persona to create the memory for.
</ParamField>

<ParamField body="agentId" type="string" optional>
  The ID of the agent to create the memory for.
</ParamField>

<ParamField body="agentReferenceId" type="string" optional>
  The reference ID of the agent to create the memory for.
</ParamField>

### Response

<RequestExample bash>
```bash
curl -X POST 'https://api.privvy.xyz/memory/create/text' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_API_KEY' \
-d '{"content": "Your plain text content here", "title": "Optional Title", "agentId": "agent123", "agentReferenceId": "agent123", "personaId": "persona123"}'
```
</RequestExample>

<ResponseExample>
```json
{
  "memory": {
    "id": "60a3e5b9f3f3a2001f3e8d1a",
    "identifier": "abc123def456",
    "autoUpdate": false,
    "settingsType": "CUSTOM",
    "type": "PlainText",
    "status": "InProgress"
  },
  "message": "Text memory created successfully"
}
```
</ResponseExample>

<ResponseField name="memory" type="object">
  The created memory object.
</ResponseField>

<ResponseField name="message" type="string">
  A success message.
</ResponseField>

<Note>
  Memory creation is subject to account limits based on subscription type.
</Note>
