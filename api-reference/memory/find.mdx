---
title: 'Find a Memory'
api: 'GET /memory/:id'
description: 'Gets a specific memory by its ID'
authMethod: "bearer"
---

This endpoint loads a specific memory by its ID for the authenticated user's account.

### Path Parameters

<ParamField path="id" type="string" required>
  The ID of the memory to get.
</ParamField>

### Response

<ResponseField name="memory" type="object">
  The memory object with the following properties:
</ResponseField>

<ResponseField name="_id" type="string">
  The unique identifier of the memory.
</ResponseField>

<ResponseField name="account" type="string">
  The account ID associated with the memory.
</ResponseField>

<ResponseField name="type" type="string">
  The type of the memory. Possible values: 'website', 'pdf', 'custom', 'github', 'gitbook', 'google_doc', 'discord_messages', 'confluence', 'notion', 'PlainText'.
</ResponseField>

<ResponseField name="identifier" type="string">
  A unique identifier for the memory.
</ResponseField>

<ResponseField name="settingsType" type="string">
  The type of settings for the memory. Possible values: 'WebsiteMemorySettings', 'PdfMemorySettings', 'CustomMemorySettings', 'GithubMemorySettings', 'GitbookMemorySettings', 'GoogleDocMemorySettings', 'DiscordMessagesMemorySettings', 'ConfluenceMemorySettings'.
</ResponseField>

<ResponseField name="autoUpdate" type="boolean">
  Indicates if the memory is set to update automatically.
</ResponseField>

<ResponseField name="settings" type="string">
  The ID of the settings object associated with this memory.
</ResponseField>

<ResponseField name="documentCount" type="number">
  The number of documents in this memory.
</ResponseField>

<ResponseField name="agent" type="string">
  The ID of the agent this memory belongs to.
</ResponseField>

<ResponseField name="isArchived" type="boolean">
  Indicates if the memory has been archived.
</ResponseField>

<ResponseField name="chunkCount" type="number">
  The number of chunks in this memory.
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the memory. Possible values: 'InProgress', 'Completed', 'Failed', 'CompletedWithErrors', 'Preparing'.
</ResponseField>

<ResponseField name="createdAt" type="string">
  The timestamp when the memory was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  The timestamp when the memory was last updated.
</ResponseField>

```bash

-H 'Authorization: Bearer YOUR_API_KEY'
```

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving the memory fails.
</ResponseField>

- HTTP 404: Returned if the memory is not found.
- HTTP 500: Returned if there's a server error while retrieving the memory.
