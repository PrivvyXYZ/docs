---
title: 'Get Memories'
api: 'GET /memory'
description: 'Retrieve memories for the authenticated user'
authMethod: "bearer"
---

This endpoint retrieves memories for the authenticated user's account.

### Query Parameters

<ParamField query="page" type="number">
  Optional. The page number for pagination. Minimum value is 1. Default is 1.
</ParamField>

<ParamField query="limit" type="number">
  Optional. The number of memories to return per page. Minimum value is 1, maximum is 100. Default is 20.
</ParamField>

<ParamField query="agent" type="string">
  Optional. Filter memories by agent ID.
</ParamField>

<ParamField query="persona" type="string">
  Optional. Filter memories by persona ID.
</ParamField>

### Response

<ResponseField name="memories" type="array">
  An array of memory objects.
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination information.
  <Expandable title="properties">
    <ResponseField name="currentPage" type="number">
      The current page number.
    </ResponseField>
    <ResponseField name="totalPages" type="number">
      The total number of pages.
    </ResponseField>
    <ResponseField name="totalCount" type="number">
      The total number of memories matching the query.
    </ResponseField>
    <ResponseField name="limit" type="number">
      The number of memories per page.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl 'https://api.privvy.xyz/memory?page=1&limit=20' \
-H 'Authorization: Bearer YOUR_API_KEY'
```

  Error message if retrieving memories fails.
