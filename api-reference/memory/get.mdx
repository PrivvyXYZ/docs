---
title: 'Get Memories'
api: 'GET /memory'
description: 'Retrieve memories for the authenticated user'
authMethod: "bearer"
---

This endpoint retrieves memories for the authenticated user's account.

### Query Parameters

<ParamField query="name" type="string">
  Optional. Filter memories by name.
</ParamField>

### Response

<ResponseField name="memories" type="array">
  An array of memory objects.
</ResponseField>

<RequestExample>
```bash
curl 'https://api.privvy.xyz/memory' \
-H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving memories fails.
</ResponseField>

- HTTP 500: Returned if there's a server error while retrieving memories.
