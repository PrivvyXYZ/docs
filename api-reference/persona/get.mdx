---
title: 'Get Personas'
api: 'GET /persona'
description: 'Gets personas for the authenticated user with pagination'
authMethod: "bearer"
---

This endpoint retrieves personas for the authenticated user's account with pagination support.

### Query Parameters

<ParamField query="page" type="number">
  Optional. The page number for pagination (minimum: 1, default: 1).
</ParamField>

<ParamField query="limit" type="number">
  Optional. The number of personas per page (minimum: 1, maximum: 100, default: 20).
</ParamField>

<ParamField query="agent" type="string">
  Optional. Filter personas by agent ID.
</ParamField>

### Response

<ResponseField name="personas" type="array">
  An array of persona objects.
</ResponseField>

<ResponseField name="totalCount" type="number">
  The total number of personas matching the query.
</ResponseField>

<ResponseField name="page" type="number">
  The current page number.
</ResponseField>

<ResponseField name="totalPages" type="number">
  The total number of pages available.
</ResponseField>

<RequestExample>
```bash
curl 'https://api.example.com/persona?page=1&limit=20' \
-H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

<ResponseExample>
```json
{
  "personas": [
    {
      "id": "60a3e5b9f3f3a2001f3e8d1a",
      "name": "Example Persona",

    }

  "totalPages": 3

```
</ResponseExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving personas fails.
</ResponseField>

<ResponseField name="context" type="string">
  Optional. Additional context for the error.
</ResponseField>

- HTTP 500: Returned if there's a server error while retrieving personas.

<ResponseExample>
```json
{
  "error": "Finding personas failed",
  "context": "Optional error context"
}
```
</ResponseExample>
