---
title: 'Get Persona by ID'
api: 'GET /persona/{_id}'
description: 'Get a specific persona by its ID'
authMethod: "bearer"
---

This endpoint loads a specific persona by its ID for the authenticated user's account.

### Path Parameters

<ParamField path="_id" type="string" required>
  The ID of the persona to get.
</ParamField>

### Response

<ResponseField name="persona" type="object">
  The retrieved persona object.
</ResponseField>

<RequestExample>
```bash
curl 'https://api.privvy.xyz/persona/60a3e5b9f3f3a2001f3e8d1a' \
-H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

<ResponseExample>
```json
{
  "persona": {
    "id": "60a3e5b9f3f3a2001f3e8d1a",
    "name": "My Persona",
    "account": "60a3e5b9f3f3a2001f3e8d1b",
    "createdBy": "60a3e5b9f3f3a2001f3e8d1c",
    "createdAt": "2023-05-18T12:00:00.000Z",
    "updatedAt": "2023-05-18T12:00:00.000Z"
  }
}
```
</ResponseExample>

### Error Responses

<ResponseField name="error" type="string">
  Error message if retrieving the persona fails.
</ResponseField>

- HTTP 500: Returned if there's a server error while retrieving the agent.
